Question 20

You have servers that run Windows Server 2022 as shown in the following table.

Name Location Description
Server1 On-premises Hosts a Microsoft SQL Server 2019 instance
Server2 Azure Contains the .NET SDK

Server2 contains a .NET app named App1.

You need to establish a WebSocket connection from App1 to the SQL Server instance on Server1. The solution must meet the following requirements:

  • Minimize the number of network ports that must be open on the on-premises network firewall.
  • Minimize administrative effort.

What should you create first?

Explanation:

Azure Relay allows you to securely expose services that run on your corporate network to the public cloud. It minimizes the number of ports that need to be open on the firewall and requires less administrative effort compared to setting up a full VPN gateway or a hybrid connection.

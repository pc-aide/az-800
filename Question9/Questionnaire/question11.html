Question 11

You have a server named Server1 that runs Windows Server and contains a file share named Share1.

You need to prevent users from storing MP4 files in Share1. The solution must ensure that the users can store other types of files in the share.

What should you configure on Server1?

Explanation:

Open the File Server Resource Manager (FSRM) on Server1.

Navigate to "File Screening Management" and select "File Screens".

Create a new file screen for Share1.

Define a file screen template or create a custom file screen to block MP4 files.

Apply the file screen to Share1.
